# DatodAPI

Aplicación web para consultar precios de carburantes en estaciones de servicio de España.
Los datos se obtienen desde la API REST del Ministerio para la Transición Ecológica y el Reto Demográfico.

Instalación:

pip install streamlit requests pandas

Uso:

streamlit run app.py

Se abrirá en el navegador en http://localhost:8501

Funcionalidades:

1. Estaciones de servicio por Comunidad Autónoma
2. Postes marítimos por provincia
3. Precios históricos por provincia, fecha y tipo de carburante
